## Getting Started

- [The Zig Programming Language](https://ziglang.org/) – official website & documentation
- [Zig Guide](https://zig.guide) – excellent structured introduction
- [Ziglings](https://ziglings.org) – learn Zig by fixing tiny broken programs (highly recommended)
- [Official Getting Started](https://ziglang.org/learn/getting-started/) – setup guide
- [Zig Build System](https://ziglang.org/learn/build-system/) – master `build.zig`

## Books

- [Learning Zig](https://www.openmymind.net/learning_zig/) – free, modern introduction (Karl Seguin)
- [Introduction to Zig](https://pedropark99.github.io/zig-book/) – project-based free book
- [Zig by Example](https://zigbyexample.com) – hands-on code examples
- [Zig Cookbook](https://cookbook.ziglang.cc/) – common tasks and best practices

## Projects & Applications

### CLI Tools & Apps
- [Ghostty](https://github.com/ghostty-org/ghostty) – blazing-fast, feature-rich terminal emulator (core written in Zig)
- [Bun](https://github.com/oven-sh/bun) – all-in-one JavaScript/TypeScript runtime, bundler, and package manager (heavily built in Zig)

### Web & Runtimes
- [Zap](https://github.com/zigzap/zap) – extremely fast HTTP server & web framework
- [Jetzig](https://github.com/jetzig-framework/jetzig) – full-featured web framework with excellent DX
- [Zinc](https://github.com/zon-dev/zinc) – secure, high-performance web framework

### Games & Graphics
- [Mach](https://github.com/hexops/mach) – modern game engine & graphics toolkit
- [zig-gamedev](https://github.com/zig-gamedev/zig-gamedev) – game development ecosystem & examples
- [raylib + Zig](https://github.com/raysan5/raylib) – simple & fast game library with first-class Zig support

### Systems & Embedded
- [TigerBeetle](https://github.com/tigerbeetle/tigerbeetle) – distributed financial accounting database (safety + performance focused)
- [MicroZig](https://github.com/ZigEmbeddedGroup/microzig) – unified embedded abstraction & HAL for microcontrollers

### Other Cool Projects
- [Flow](https://github.com/neurocyte/flow) – programmer’s text editor written entirely in Zig
- [Tetris](https://github.com/andrewrk/tetris) – classic Tetris clone by Zig’s creator Andrew Kelley

## Tools & Utilities

- [Zig Language Server (ZLS)](https://github.com/zigtools/zls) – official LSP for VS Code, Neovim, etc.
- [zigup](https://github.com/marler8997/zigup) – effortless Zig version manager
- [Zigmod](https://github.com/nektro/zigmod) – package manager for Zig
